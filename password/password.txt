 Task 3 - Password Generator

 Description:
A Python-based password generator that creates strong, secure, and random passwords based on user preferences. The user can choose the length and whether to include letters, numbers, and special characters.

 Features:
- Generates strong random passwords
- User selects:
  - Password length
  - Inclusion of uppercase/lowercase letters
  - Inclusion of digits
  - Inclusion of special characters
- Easy-to-use command-line interface
- Prevents weak password generation

 Technologies Used:
- Python
- `random` module
- `string` module

 How to Run:
```bash
python password_generator.py
